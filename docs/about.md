# About

You can create a table using dashes and piping.
You can use syntax to add formatting.

| Feature       | Supported | Notes                          |
|---------------|-----------|--------------------------------|
| Headings      | Yes       | Use `#` or `##`                |
| Tables        | Yes       | Simple pipe-based format       |
| Code blocks   | Yes       | Use triple backticks (` ``` `) |
